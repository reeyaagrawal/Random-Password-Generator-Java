# Random Password Generator (Java)

This is a Java program that generates strong random passwords.

## Features
- Generates multiple passwords
- Custom password length
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Digits
- Uses shuffle for better randomness

## How it works
- Ensures at least one uppercase, lowercase, and digit
- Fills remaining characters randomly
- Shuffles characters to avoid predictable patterns

## Sample Output
A9kLp2
xT3mQa
